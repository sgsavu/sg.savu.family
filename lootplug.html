Ad Rewards Loot Plug Rewards

Mobile Android Application

Developed a mobile android application that rewards users a virtual currency by watching 15-30 seconds advertisements. The user can then redeem rewards such as Amazon gift cards etc. by using the virtual currency they have obtained. This is possible from the revenue the users generate while watching ads. The prices for the Amazon cards are thoroughly calculated so that a profit is generated within the cycle.

 

 

Although the app was aimed specifically for Roblox users, its scope has expanded into a more general use by a broader audience.

Anyone that wishes to be rewarded for their watch time can earn fantastic rewards while generating an income for the project developers.
